# 🌍 Smart Tourist Safety Monitoring System

## 📌 Overview

The **Smart Tourist Safety Monitoring and Incident Response System** is a web-based frontend application designed to enhance tourist safety using modern technologies like AI, geofencing, and blockchain concepts.

This project provides real-time monitoring, risk detection simulation, and emergency alert features to ensure a safer travel experience.

---

## 🚀 Features

* 📍 Live Location Tracking (Geolocation API)
* 🛡 Safety Status Monitoring (Safe / Danger)
* 🤖 AI Risk Detection (Simulated Logic)
* 🚨 SOS Emergency Alert Button
* 📡 Geofencing Simulation (Danger Zone Detection)
* 🔔 Notification Alerts (Toast Messages)
* 🎨 Modern UI (Glassmorphism + Gradient Design)
* 📱 Responsive Design (Mobile + Desktop)

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (Vanilla JS)
